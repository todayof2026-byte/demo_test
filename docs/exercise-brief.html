4.Project Description

Four core functions must be developed:

  • Authentication (login)
  • Search with price condition - searchItemsByNameUnderPrice
  • Add items to cart - addItemsToCart
  • Assert cart total - assertCartTotalNotExceeds

4.1Search Function with Price Condition

Returns up to N item URLs whose price is less than or equal to maxPrice.

Signature (TypeScript / Playwright example)

// Returns up to N item links whose price is <= maxPrice
async function searchItemsByNameUnderPrice(
    query: string,
    maxPrice: number,
    limit = 5
): Promise<string[]>

Behavior

  • Perform a search using the query parameter.
  • If a price filter is available on the page, use it (min/max) to narrow down the results.
  • Use XPath to extract the first limit items (e.g. five) whose price is less than or equal to maxPrice.
  • Edge case - fewer than 5 items on the current page:
    • If paging is available (a "Next" button or page links), navigate to the next page and continue collecting items until reaching 5, or until the pages run out.
    • If paging is not available, return the items that were found (even if fewer than 5).
  • Return value: an array of URLs - up to 5 results that match the price condition.
  • If fewer items were found, return whatever is available (returning 0 items is also valid).

Usage example

const urls = await searchItemsByNameUnderPrice("shoes", 220, 5);

4.2Function That Adds Items to the Cart

Signature

async function addItemsToCart(urls: string[]): Promise<void>

Behavior

  • Iterate over each URL in a loop and open the item page.
  • If variants must be selected (size / color / quantity), pick values randomly from the available options.
  • Click "Add to cart".
  • Return to the search screen / search tab.
  • Save a screenshot and log entry for every item that was added.

4.3Function That Validates the Cart Total

Signature

async function assertCartTotalNotExceeds(
    budgetPerItem: number,
    itemsCount: number
): Promise<void>

Behavior

  • Open the shopping cart.
  • Read the subtotal / total amount as displayed on the site.
  • Calculate the threshold: budgetPerItem * itemsCount.
  • Assert that the total does not exceed the threshold.
  • Save a screenshot / trace of the cart page.

Full scenario example

  • Call searchItemsByNameUnderPrice("shoes", 220, 5) → receive up to 5 URLs.
  • Call addItemsToCart(urls) → add all of them to the shopping cart.
  • Call assertCartTotalNotExceeds(220, urls.length) → assert that the cart total ≤ 220 × number of items.
